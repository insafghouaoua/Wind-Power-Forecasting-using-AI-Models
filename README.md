# Wind-Power-Forecasting-using-AI-Models

## Overview
An advanced AI-driven system designed to predict wind energy production using a multi-model approach. This project compares 15 different architectures—ranging from classical statistics to Quantum Machine Learning—tested on real-world datasets from Adrar (Algeria) and Yalova (Turkey).

## Key Features
- Multi-Model Comparison: Evaluation of 15 models (Statistical, Boosting, Neural Networks, Hybrid, and Quantum AI).

- Cross-Geographic Testing: Validation using two distinct climate contexts (Algeria & Turkey).

- End-to-End Pipeline: Comprehensive workflow including anomaly detection (Isolation Forest, SVM) and meteorological data enrichment.

- Quantum Integration: Implementation of a Hybrid Quantum Neural Network (QNN) using TensorFlow Quantum.

- Interactive Dashboard: A modern web interface to visualize results and simulate forecasts.

## Tech Stack
- Languages: Python, JavaScript (ES6+), HTML5/CSS3

- AI/ML: XGBoost, LightGBM, Scikit-learn, TensorFlow, TensorFlow Quantum

- Data: Pandas, NumPy, Meteostat API

- Visualization: Chart.js, Matplotlib, Seaborn
## Project Content

This repository is organized to ensure full reproducibility:

- Jupyter Notebooks: Step-by-step implementation of preprocessing, feature engineering, and modeling.

- Datasets: Sampled time-series records (10–15 min intervals) from Adrar and Yalova.

- Web Interface: Source code for the interactive portfolio and forecasting simulator.

- Evaluation: Detailed multi-metric reports (RMSE, MAE, R², MAPE).
